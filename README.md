# FAIRsharing.org

Welcome to the new FAIRsharing.org website.

## Compiles and hot-reloads for development
```
npm run serve
```

## Test with vue-cli & generate coverage
```
npm run test:unit
```

## Generate the documentation locally
```
npm run doc:full
```
